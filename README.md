# Online Exam Management System

## 📌 Project Description
Online Exam Management System is a web application developed using Java, JSP, Servlets, and MySQL. It allows administrators to manage exams, questions, and results while students can take exams online.

## 🛠 Technologies Used
- Java
- JSP
- Servlets
- MySQL
- HTML
- CSS
- Apache Tomcat
- NetBeans IDE

## ✨ Features
- Admin Login
- Student Login
- Create Exam
- Manage Questions
- Take Online Exam
- Automatic Result Generation

## 🗄 Database
Import the `data_base.sql` file into MySQL.

## ▶ How to Run
1. Import project in NetBeans
2. Configure MySQL database
3. Run on Apache Tomcat Server
